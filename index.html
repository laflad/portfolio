See it, Hear it, Report it!

Drawing attention with color. Some information was removed for proprietary purposes. This project was created to empower employees to report suspecious activity to keep the workplace a safe environment.

Dinosaur

This visual aims at informing employees that they should report any situation where another individual seems agitated or aggressive towards others.


Fish and Bait

This illustration informs employees that they should report any suspicious package or items being left unattended. The bait is a metaphore for a suspicious package.


Chatter Teeth

Similar to the dinosaur image, this one informs employees that they should report if they see someone raising their voice to other associates.


UPO Stealing a Package

This image illustrates a UFO stealing a package to inform our employees to report if they witness theft.
